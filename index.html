Comandos para trabajar con un repositorio

    git init
    Crea un repositorio local en la carpeta en la que se esta posicionado
    git log
    Permite revisar la actividad en el repositorio
    git add "nombre-archivo"
    Agrega los archivos seleccionados del workin storage al stagging storage
    git commit -m "mensaje que describe la modificaci√≥n"
    Agrega los archivos del stagging storage al repositorio
    git branch "nombre-rama"
    Permite crear un rama en el repositorio con el nombre indicado
    git checkout "nombre-rama"
    Permite moverse a la rama con el nombre indicado
    git checkout -b "nombre-rama"
    Permite crear y moverse a la rama con el nombre indicado
